This system prioritizes execution over motivation.
This repository manages a daily 5-minute habit system.

- 12 habits per day
- Each habit is designed to be completed within 5 minutes
- Focus on consistency over intensity

---
## ルール
- 1週 = 6日(実行) + 1日(予備/休憩)
- スコア: 1習慣で85%の達成を目指す(71/85以上)
- ×の日も必ず記録する (未達は失敗ではなく観測)

---
## 1日5分と1年
1年 = 72日 × 5
72日 = 12半日 × 12
12半日 = 半日 × 12
半日 = 1時間 × 12
1時間 = 5分 × 12
